#### 아나콘다(Anaconda) 사용법 및 가상환경 관리하는법
위 참조 동영상을 보고 정리해보자. \\

1. 버전 확인하기( $ conda --version)

2. 프로젝트 만들기 ($ conda create -n hello_conda python=3.6)
-- -n == --name과 같은 것임.

3. 가상환경 리스트 확인하기 ($ conda env list)

4. 가상환경 지우기 ($ conda remove -n hello_conda all)
